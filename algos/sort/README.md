

## 排序算法总结


|     排序算法    | 平均时间复杂度 |             辅助空间              |    稳定性                        |
| ------------   |    ---      | -------------------------------  | ------------------------------- |
|     冒泡排序    |  O(n^2)      |             O(1)                |           稳定                   |
|     选择排序    |  O(n^2)      |             O(1)                |           不稳定                 | 
|     插入排序    |  O(n^2)      |             O(1)                |           稳定                   | 
|     归并排序    |  O(nlogn)    |             O(n)                |           稳定                   | 
|     快速排序    |  O(nlogn)    |             O(nlogn)            |           不稳定                 | 
|     堆排序      |  O(nlogn)    |             O(1)                |           不稳定                | 
|     希尔排序    |  O(nlogn)    |             O(1)                |           不稳定                | 
|     计数排序    |  O(n + k)    |             O(n + k)            |           稳定                  | 
|     选择排序    |  O(n + k)    |             O(n + k)            |           稳定                  | 
|     选择排序    |  O(n*k)      |             O(n + k)            |           稳定                  |
